数组方法

  • map
    根据数组生成相同长度的新数组

  • filter
    根据数组筛选符合条件的内容组成新的数组，新数组长度可能会变，但是新数组中的结构不会改变

  • forEach
    代替for循环
  • some
    数组中是否有某个值符合条件
  • every
    数组中是否所有值都符合条件
  • reduce
    累加
  • reduceRight
    从右边累加

  • find
    根据条件去数组中找到对应的符合条件的值，只会找到第一个

  • indexOf

  • lastIndexOf
