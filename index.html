npm常用命令

初始化

npm init

初始化的目的就是为了生成package.json

package.json

该文件中保存了当前项目的信息 dependencies devDependencies 一个是保存了需要的模块，另外是开发环境中需要的模块

在安装时使用npm install 模块名 --save可以将模块名保存在dependencies

使用npm install 模块名 --save-dev可以将模块名保存在devDependencies

package.json中保留的是模块清单，不管在哪个位置，只要有package.json，我们都可以直接安装所有清单上的模块

安装（下载模块）

npm install -g 模块名 // 全局安装

npm install 模块名 –save

npm install 模块名 –save-dev

npm install // 直接根据package中所需要的模块进行自动安装

删除模块

npm uninstall 模块名 –save
npm uninstall 模块名 –g

yarn

facebook和一些大厂合作退出一个nodejs包管理工具，下载更快

npm install -g yarn

安装

yarn global add 模块名

yarn add 模块名

yarn add 模块名 –dev

yarn

删除模块

yarn remove 模块名
